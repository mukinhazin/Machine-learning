# Machine-learning
Study about the types of machine learning

At first point, every information contains a *WEIGHT* and these *WEIGHT* can be high, medium or low depending of the information.
After this, an equation will be aplicated to generate a new information. About it, the new information contains a value and this value can be used to be the anwser or, in other cases, to be a new information with the others, helping the model to adjust and trying to strike the goal.

## The first model of learning is *SUPERVISIONED LEARNING*

> In the **SUPERVISIONED LEARNING,** the model knows the awnser and calculate the informations until find the better weights for every information.

## The second model of learning is *UNSUPERVISIONED LEARNING*

>In the **UNSUPERVISIONED LEARNING,** the model doesn´t know the awnser. So, it works gathering similar informations.

## The third model of learning is *REINFORCEMENT LEARNING*

>In the **REINFORCEMENT LEARNING,** the model doesn´t have an awnser or informations. The model will create, analyse and label the experience. Here, the model doesn´t have a nagative or positive information, it works with experience.
